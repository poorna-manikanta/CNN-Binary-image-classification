# CNN Binary Image Classification

A simple end-to-end **binary image classification** web application built using **TensorFlow (MobileNet)** and **Streamlit**.

## Overview
This project demonstrates how a deep learning image classification model can be deployed as an interactive web application for real-time predictions.

## Model
- Architecture: MobileNet (Transfer Learning)
- Task: Binary Image Classification
- Input Size: 224 Ã— 224

## Web Application
- Built using Streamlit
- Upload an image and get instant prediction results

## Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- PIL

## Deployment
The application is deployed on **Streamlit Cloud** and runs directly from this GitHub repository.
